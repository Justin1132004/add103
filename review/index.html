LittleBigPlanet

LittleBigPlanet is a game released in for the Sony Playstation. The game is a sandbox platformer that allows users to create and submit levels through a robust level editor.

Gameplay

The game stars the titular Sackboy, a sentient doll that the user must guide through official and user made platforming levels. There is a lives system in the form of checkpoints that the player must reach, the number of lives depends on how many were specified for that checkpoint by the creator of the level.
